Instructions:

Test 3: Drawer State Management

Instructions:
  1. Open notification drawer on desktop
  2. Switch to mobile size using controls above
  3. Verify drawer closes automatically

Expected Behavior: Drawer state should be properly managed across all screen size transitions.
